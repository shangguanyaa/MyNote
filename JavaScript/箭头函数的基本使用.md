# 箭头函数基本使用



## 定义函数的方式



### 1. function

```javascript
const aaa = function () { }
```

### 2. 对象字面量中定义函数

```js
const obj = {
	bbb: function() { },
	aaa: function() { }
}
```

### 3. ES6中的箭头函数

#### 3.1 参数问题

```js
// 没参数没返回值
const ccc = () => {}
```

##### 3.1.1 放入两个参数

```js
// 正常写法
const sun = (num1, num2) {
	return num1 + num 2
}
```

##### 3.1.2 放入一个参数

```js
// 正常写法
const power = (num) => {
	return num * num
}
```
